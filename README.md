# prototipo-ionic-vue


## Prerequisites
```
Install NVM
nvm install node (v 12 or higher)
Clone repo
CD ionic-poc-mobile-hybrid
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
